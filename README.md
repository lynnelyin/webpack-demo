## 技术亮点：

### 基于webpack性能优化自己搭建了一套脚手架，区分开发环境和生产环境

### 手动实现loader、plugin

### 手动实现迷你bundler（入口文件分析，依赖图谱，代码生成）


## 生产环境打包

`npm run build`

## 开发环境打包

`npm run dev`

## loader

`cd make-loader`

`npm run build`

## plugin

`cd make-plugin`

`npm run build`

## bundler

`cd bundler`

`node bundler.js`
